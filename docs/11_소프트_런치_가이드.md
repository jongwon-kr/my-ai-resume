# CloneCV 소프트 런치 가이드

[프로덕션 체크리스트](./10_프로덕션_체크리스트.md) Go 판정 후, **10~30명** 규모로 반응을 검증합니다.

## 1. 대상

- IT 취업·이직 준비 지인
- 포트폴리오를 링크로 공유하는 개발자 지인
- 피드백을 솔직히 줄 수 있는 사람

## 2. 초대 메시지 (예시)

> CloneCV 베타를 써봐 주실 수 있을까요?  
> 이력서를 입력하면 `/@슬러그` 주소에서 AI와 대화하는 프로필이 만들어집니다.  
> 예시: https://my-ai-resume-alpha.vercel.app/@kimdev  
> 가입 후 15분이면 발행까지 가능해요. 불편한 점은 편하게 알려주세요.

## 3. 참여자에게 요청할 액션

1. 가입 → slug 설정 → 이력서 최소 입력 → **발행**
2. 본인 `/@slug` 링크를 1곳 이상 공유 (지원서, SNS, 지인에게)
3. 방문자(본인 또는 친구)가 채팅 1회 이상
4. (선택) 두 번째 프로필 생성 — 이직용/포트폴리오용 등

## 4. 수집 지표 (Supabase Dashboard)

| 지표 | 테이블/위치 | 의미 |
|------|-------------|------|
| 가입 수 | Auth → Users | 유입 |
| 발행 프로필 | `profiles.status = published` | 핵심 전환 |
| 조회수 | `profile_daily_stats`, `view_count` | 링크 공유 효과 |
| 채팅 세션 | `chat_sessions` (visitor) | AI 가치 검증 |
| 문의 | `inquiries` | human fallback 필요도 |

Sentry가 연결되어 있으면 에러율도 함께 확인합니다.

## 5. 피드백 수집

- 구글 폼 또는 Notion 1페이지 (5문항 이내 권장)
  - 발행까지 걸린 시간?
  - 채팅 답변 품질 (1~5)?
  - 다시 쓸 의향?
  - 가장 불편했던 점?
  - 유료라면 얼마까지?

## 6. 종료 기준 (2주)

| 신호 | 다음 단계 |
|------|-----------|
| 발행률 50%+ , 채팅 1회+ 30%+ | [성장 가이드](./12_성장_가이드.md)로 확대 |
| 발행률 낮음, UX 불만 다수 | 온보딩·빌더 개선 후 재베타 |
| Gemini 비용/에러 급증 | rate limit·모델 정책 조정 |

## 7. 주의

- 베타 기간 **일반 공개 SNS 대량 홍보는 보류** (비용·스팸 리스크)
- 신고(`/admin`) 모니터링 주 1회
- Upstash·Gemini 쿼터 주 1회 확인
