# CloneCV 기능 분해도 (Feature Breakdown Structure)

버전: v1.0 | 작성일: 2026-07-05

```
CloneCV (MVP)
│
├── 1. 인증/계정 (Auth)
│   ├── 1.1 회원가입 (이메일/Google OAuth)
│   ├── 1.2 로그인/로그아웃
│   ├── 1.3 비밀번호 재설정
│   └── 1.4 고유 ID(슬러그) 설정 및 중복검사
│
├── 2. 이력서 빌더 (Resume Builder)
│   ├── 2.1 멀티스텝 폼 UI
│   │   ├── 2.1.1 기본정보 입력
│   │   ├── 2.1.2 기술스택 입력 (태그 자동완성)
│   │   ├── 2.1.3 프로젝트 입력 (최대 3개, 트러블슈팅 구조화)
│   │   └── 2.1.4 미리보기 및 발행
│   ├── 2.2 자동저장/임시저장 (Draft)
│   └── 2.3 유효성 검사 및 입력 가이드
│
├── 3. 시스템 프롬프트 엔진
│   ├── 3.1 데이터 → 프롬프트 템플릿 조합 로직
│   ├── 3.2 가드레일 지시문 삽입
│   └── 3.3 프롬프트 버전 관리 (재발행 시 갱신)
│
├── 4. 공개 프로필 페이지
│   ├── 4.1 이력서 뷰 (좌측 패널)
│   ├── 4.2 채팅 뷰 (우측 패널)
│   ├── 4.3 반응형/모바일 탭 전환
│   ├── 4.4 OG 메타태그/이미지 자동생성
│   ├── 4.5 비공개 모드 처리
│   └── 4.6 조회수 카운팅
│
├── 5. AI 대화 엔진 (RAG Chatbot)
│   ├── 5.1 LLM 호출 및 스트리밍 응답
│   ├── 5.2 대화 히스토리 컨텍스트 관리
│   ├── 5.3 범위 밖 질문 방어 (가드레일)
│   ├── 5.4 응답 후처리 필터 (민감정보/악성 콘텐츠)
│   ├── 5.5 대화 로그 저장
│   └── 5.6 레이트 리미팅
│
├── 6. 소유자 대시보드
│   ├── 6.1 프로필 관리(수정/비공개/링크복사)
│   ├── 6.2 대화 로그 열람
│   └── 6.3 통계(조회수/세션수 차트)
│
├── 7. 성장/바이럴 장치
│   ├── 7.1 워터마크 CTA
│   ├── 7.2 소셜 공유 버튼(카카오톡/링크복사/X)
│   └── 7.3 랜딩페이지 + 예시 갤러리
│
├── 8. 모더레이션 (관리자)
│   ├── 8.1 신고하기 (방문자)
│   └── 8.2 관리자 대시보드(신고 처리)
│
└── 9. 공통 인프라
    ├── 9.1 DB 스키마/RLS 정책
    ├── 9.2 배포 파이프라인(CI/CD)
    ├── 9.3 로깅/모니터링
    └── 9.4 비용 모니터링 (LLM API 사용량)
```

## 개발 착수 순서 제안 (Cursor 작업 단위와 매핑)

| 순서 | 모듈                               | 예상 소요 |
| ---- | ---------------------------------- | --------- |
| 1    | 9.1 DB 스키마 + 1. 인증            | 3~4일     |
| 2    | 2. 이력서 빌더                     | 5~6일     |
| 3    | 3. 시스템 프롬프트 엔진            | 2일       |
| 4    | 4. 공개 프로필 페이지 (레이아웃) | 3일       |
| 5    | 5. AI 대화 엔진                    | 4~5일     |
| 6    | 6. 대시보드 + 7. 성장장치          | 3~4일     |
| 7    | 8. 모더레이션 + QA/버그픽스        | 2~3일     |
