## 3. 최근 변경 요약 (2026-07-07 ~ 07-15)

- 한국 시장 맞춤: PII 토글, 문의하기, 모의 면접, FAQ 루프, OG/카카오 통일
- 채팅 인사말 단순화 (프로필 하이라이트 문구 제거)
- RSS/GitHub 외부 소스 **제거** → 외부 링크 자유 입력으로 대체
- 랜딩 예시: 조회수 1위 실제 프로필 → **`@kimdev` 가상 프로필** 고정
- 빌더 9단계: 학력·자격·활동 분리, 섹션 드래그 순서
- **다중 프로필(3개)·프로필 라벨** — 대시보드 전환/생성 UI
- **MVP 보안 hardening** — RLS, PII strip, API rate limit, Sentry(선택)
