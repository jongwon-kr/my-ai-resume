# CloneCV 기술 스택 및 필요 서비스 정리

버전: v1.0 | 작성일: 2026-07-05

MVP를 1개월 내에 완성하기 위해 **직접 구축보다 관리형 서비스(BaaS/SaaS) 조합**을 우선했습니다.

## 1. 핵심 스택

| 영역                  | 선택                           | 이유                                                                                                             | 예상 비용(초기)                     |
| --------------------- | ------------------------------ | ---------------------------------------------------------------------------------------------------------------- | ----------------------------------- |
| 프론트엔드 프레임워크 | Next.js 16 (TypeScript)        | SSR/SEO 필요(퍼블릭 프로필), Cursor와 궁합 좋음, Vercel 배포 최적화                                              | 무료                                |
| 스타일링              | Tailwind CSS + shadcn/ui       | 빠른 UI 구축, 커스터마이징 용이                                                                                  | 무료                                |
| 호스팅/배포           | Vercel                         | Next.js 네이티브 지원, 프리뷰 배포, 서버리스 함수                                                                | 무료(Hobby) → 성장 시 Pro $20/월    |
| 인증 + DB + Storage   | Supabase                       | 인증/DB/RLS/스토리지 통합, 무료 티어로 MVP 충분                                                                  | 무료(Free tier) → Pro $25/월        |
| LLM                   | Google Gemini API              | Claude API는 유료 플랜 필요; MVP는 `gemini-2.5-flash` 무료 티어 사용 ⚠️ **[변경: Claude → Gemini, Pro → Flash]** | 사용량 종량제 (레이트리밋으로 통제) |
| 레이트리밋/캐시       | Upstash Redis                  | 서버리스 친화적, 종량제, 설정 간단                                                                               | 무료 티어로 MVP 충분                |
| 폼 관리               | React Hook Form + Zod          | 멀티스텝 폼 유효성 검증에 적합                                                                                   | 무료(오픈소스)                      |
| 에러 모니터링         | Sentry                         | 프론트/백엔드 통합 모니터링                                                                                      | 무료 티어                           |
| 분석                  | Vercel Analytics 또는 PostHog  | 트래픽/사용자 행동 추적, 바이럴 효과 측정                                                                        | 무료 티어                           |
| 이메일 발송           | Resend 또는 Supabase 기본 SMTP | 비밀번호 재설정 등 트랜잭션 메일                                                                                 | 무료 티어                           |
| 소셜 공유             | 카카오 SDK (JS SDK)            | 카카오톡 공유 카드 지원                                                                                          | 무료                                |

## 2. 계정/키 준비물 (개발 시작 전 체크리스트)

- [ ] GitHub 저장소 생성
- [ ] Vercel 계정 연결 (GitHub 연동)
- [ ] Supabase 프로젝트 생성 (URL, anon key, service role key 발급)
- [ ] Google AI Studio에서 Gemini API 키 발급 (aistudio.google.com) — Gemini Pro 모델 사용 가능 여부 확인
- [ ] Upstash Redis 인스턴스 생성 (REST URL/Token)
- [ ] 카카오 디벨로퍼스 앱 등록 (JS 키, 공유 API 활성화)
- [ ] 도메인 구매 및 Vercel 연결 (예: clonecv.com)
- [ ] Sentry 프로젝트 생성 (선택)

## 3. 환경변수 목록 (.env.local 예시)

```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=       # 서버 전용, 절대 클라이언트 노출 금지
GEMINI_API_KEY=                  # 서버 전용 (Google AI Studio 발급)
UPSTASH_REDIS_REST_URL=
UPSTASH_REDIS_REST_TOKEN=
NEXT_PUBLIC_KAKAO_JS_KEY=
NEXT_PUBLIC_SITE_URL=https://clonecv.com
SENTRY_DSN=                      # 선택
```
